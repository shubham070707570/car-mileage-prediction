# MACHINE LEARNING MULTICLASS REGRESSION car-mileage-prediction 
This repository contains a machine learning project that predicts car mileage (MPG) using  multiple linear regression techniques in Python. The project includes data preprocessing, model training, evaluation, and visualization using popular libraries like pandas, scikit-learn, and matplotlib.
This project builds machine learning models to predict car mileage (MPG) using car features such as weight, horsepower, and model year. It includes 
:
